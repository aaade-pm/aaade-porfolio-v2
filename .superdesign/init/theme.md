# Theme

Full tokens: `styles/globals.css` (`:root`, `@theme inline`, `@layer base`, `@layer components`, `@layer utilities`).

Summary:

- **Background:** `--background` / `bg-background`
- **Accent:** `--primary` / `text-primary`, `bg-primary`, `border-primary`
- **Fonts:** `--font-sans` (Satoshi), `--font-display` (Cabinet Grotesk)
- **Section spacing:** `--spacing-section`, `--spacing-section-sm`

No separate `tailwind.config.ts`; Tailwind v4 uses CSS-first config in `globals.css`.
